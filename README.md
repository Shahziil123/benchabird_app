## 📖 Managing Your First Bird Show

Managing a show requires a logical approach. Follow this guide to handle entries and results.

### 📝 Setting Up An Event
Begin by creating a new show file. Select File from the top menu, then choose New Show. Enter the name, date, and location of the event. This information appears on all reports later. The software creates a primary database file for this event.

### 🔢 Registering Exhibitors
Store information about people who bring birds. Select the Exhibitors tab. Click Add Exhibitor. Enter the name, contact details, and club information. You save changes for every person. The system remembers these people for future shows.

### 🦜 Creating Entries
Assign birds to specific classes. Select the Entries tab. Click Create Entry. Choose an exhibitor from your saved list. Enter the bird details, such as species, age, and ring number. The program assigns a cage number automatically. You can print physical cage tickets using the Print button on this screen.

### 🏆 Recording Results
Enter show outcomes once judging finishes. Go to the Results tab. Locate the entry by cage number. Select the placement or tier. The software saves these results to the database immediately.

## 📊 Generating Reports

The software creates documents for your archives. Click the Reports tab to see available options. 

*   **Entry List:** Shows all birds registered for one event.
*   **Exhibitor Summary:** Lists all entries by person.
*   **Placement Report:** Shows winners by class.
*   **Cage Labels:** Formats data for printing adhesive labels.

Choose the report you need, then click Generate. The system creates a file you can print or save as a document.

## 🔍 Frequently Asked Questions

### Does the software need the internet?
No. All features work offline. The application stores data directly on your Windows hard drive. You do not need a connection to start, run, or close a show.

### How do I save my work?
The program saves data automatically when you enter information. You do not need to click a manual save button. Close the program whenever you finish your session. Your data remains safe on your computer.

### Can I move my data to another computer?
Yes. Find the file in your documents folder that holds your show database. Copy this file to a USB drive or move it over a local network. Open the file on the new computer using the Benchabird application.

### Why does Windows show a security warning?
Windows protects users from unknown programs. Since this is an open-source project, Windows may prompt you to confirm you trust the file. Click More info and Run anyway to proceed.

## 📁 Data Management Policy

The software uses a local database system. Benchabird manages these files inside your user folder. Do not delete or rename files in the application folder. Changing these files may cause errors or the loss of show records. Keep a backup of your folder on a separate drive or cloud storage to prevent data loss.

## 🆘 Troubleshooting Common Issues

If the program fails to start, verify your Windows version. Ensure you have the latest updates for your system. If the window appears empty, check that you have created or selected a show file first. Restart the application if you encounter unexpected behavior. Always perform a fresh download from the link if the file appears damaged.
